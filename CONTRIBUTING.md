# Contributing to RedSec Learn

Thanks for helping improve RedSec Learn. Keep contributions focused on clear, authorized, beginner-friendly red team education.

## Guidelines

- Write for learners who are building fundamentals.
- Keep examples scoped to local labs, owned systems, CTFs, or intentionally vulnerable training targets.
- Avoid instructions that enable unauthorized access, stealth, persistence, or disruption.
- Add new learner-facing pages to `docs.json` when they are ready to appear in the navigation.

## Local Development

Run `pnpm install`, then `pnpm run dev` to preview the site at `http://localhost:3333/`.
