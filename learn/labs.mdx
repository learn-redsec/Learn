## Reference Material

### Methodology
- [MITRE ATT&CK Framework](https://attack.mitre.org) — the definitive catalog of adversary tactics, techniques, and procedures.
- [PTES (Penetration Testing Execution Standard)](http://www.pentest-standard.org) — methodology for professional engagements.
- [OWASP Testing Guide](https://owasp.org/www-project-web-security-testing-guide/) — comprehensive web application security testing reference.

### Cheatsheets
- [PayloadsAllTheThings](https://github.com/swisskyrepo/PayloadsAllTheThings) — massive collection of payloads and bypass techniques.
- [GTFOBins](https://gtfobins.github.io) — Unix binaries that can be used for privilege escalation.
- [LOLBAS](https://lolbas-project.github.io) — Living Off the Land Binaries and Scripts for Windows.
- [HackTricks](https://book.hacktricks.xyz) — comprehensive red team reference organized by technique.
- [Reverse Shell Generator](https://www.revshells.com) — generate reverse shell one-liners for any language.

### Learning
- [PortSwigger Web Security Academy](https://portswigger.net/web-security) — free web app security course with labs.
- [TCM Security Courses](https://academy.tcm-sec.com) — affordable, practical red team courses.
- [Sektor7 Malware Development](https://institute.sektor7.net) — advanced Windows offensive development.
- [0xdf Walkthrough Blog](https://0xdf.gitlab.io) — detailed HTB machine walkthroughs for learning methodology.
